## Key Features Implemented

- **100% Native Architecture:** Developed entirely using Flutter framework components without relying on external packages.
- **Custom Responsive Scaling:** Implemented a robust `BuildContext` extension (`responsiveHeight`) based on a benchmark layout grid. This ensures pixel-perfect UI rendering and proportional spacing across all Android & iOS screen dimensions.
- **Optimized Performance:** Smooth custom animations tailored with standard curves and dynamic scaling to prevent render-lag.
- **Strict Guidelines Adherence:** Built with the exact required App Name, Package Name structure, and design configurations specified in the assignment briefing.
